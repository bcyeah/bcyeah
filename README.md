積土成山，風雨興焉；積水成淵，蛟龍生焉；積善成德，而神明自得，聖心備焉。

故不積跬步，無以至千里；不積小流，無以成江海。

騏驥一躍，不能十步；駑馬十駕，功在不捨。

鍥而捨之，朽木不折；鍥而不捨，金石可鏤
